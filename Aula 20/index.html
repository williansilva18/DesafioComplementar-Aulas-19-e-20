Exercicio 1

Construa uma lista de tarefas:

  • Adicione um campo para o usuário inserir um nome e idade e um botão para “enviar”;

  • Quando o usuário clicar no botão “enviar”, insira na lista pra nomes e lista pra idade;

  • Insira um botão “visualizar” para exibir todos os itens das listas no console ou alert;
