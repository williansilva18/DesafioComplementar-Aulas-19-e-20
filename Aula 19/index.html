Desafio Complementar - Aula 19


Exercicio 1

Crie algumas variáveis no seu arquivo js:

  • A primeira receberá o seu nome completo

  • A segunda e a terceira receberão números

  • A quarta variável receberá a soma dos dois números definidos anteriormente

  • A quinta variável receberá a multiplicação dos mesmos números

  • Usando um alert() exiba o resultado da soma na tela

  • Usando o console.log() exiba seu nome e o resultado da multiplicação no terminal


    • Exercicio 2

    No mesmo código criado anteriormente insira 2 funções:

  • Uma função para fazer a subtração de 2 números. Para casos em que o primeiro número é menor que o segundo, crie uma condição que retorne a mensagem “Não foi possível subtrair”.

  • Uma função para fazer a divisão de 2 números. Para casos em que o segundo número é menor que zero, crie uma condição que retorne a mensagem “Não foi possível dividir”.

  • Faça alguns testes chamando sua função com diferentes valores para garantir seu funcionamento.

    • Exercicio 3

  • Crie um objeto que represente uma fruta com as seguintes propriedades: nome, cor, peso e o tipo (nesse caso, o valor do tipo será “fruta”).

  • Usando o console.log exiba no terminal o objeto que foi criado.

    • Exercicio 4

  • Crie um array com pelo menos 5 objetos em que cada objeto é um alimento que pode ser uma fruta, uma verdura ou um legume.

  • Cada alimento deve ter 4 propriedades: nome, cor, peso e o tipo (fruta, verdura ou legume).

  • Usando o console.log exiba no terminal o array de alimentos que foi criado.

    • Exercicio 5

    Usando os dados inseridos anteriormente:

  • Crie uma função que exiba no terminal apenas o nome de cada uma das frutas. Use o FOREACH para percorrer o array

  • Crie uma função que mostre no console a quantidade de alimentos de cada tipo.

  • Crie uma variável que funcionará como um contador para cada tipo

  • Use o FOREACH para percorrer o array

  • Use o IF para verificar o tipo do alimento

  • Some 1 no contador referente ao tipo analisado
