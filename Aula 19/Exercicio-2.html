Verifique o exercicio no console


No mesmo código criado anteriormente insira 2 funções: Uma função para fazer a subtração de 2 números. Para casos em que o primeiro número é menor que o segundo, crie uma condição que retorne a mensagem “Não foi possível subtrair”. Uma função para fazer a divisão de 2 números. Para casos em que o segundo número é menor que zero, crie uma condição que retorne a mensagem “Não foi possível dividir”. Faça alguns testes chamando sua função com diferentes valores para garantir seu funcionamento.
